## 10. 프론트엔드 UI/UX 설계

### 10.1 메인 페이지 (/)

**레이아웃**
```
┌─────────────────────────────────────────┐
│  PC Build Advisor                   🔧  │
├─────────────────────────────────────────┤
│                                         │
│  "어떤 PC가 필요하신가요?"              │
│  ┌─────────────────────────────────┐   │
│  │ 배틀그라운드를 렉 없이 돌리고   │ ▶ │
│  │ 싶어요. 예산은 150만원          │   │
│  └─────────────────────────────────┘   │
│                                         │
│  [🔍 예제 1] [💻 예제 2] [🎮 예제 3]   │
│                                         │
└─────────────────────────────────────────┘
```

**기능**
- 채팅 스타일의 자연어 입력
- 이전 검색 이력 (세션 저장)
- 인기 게임/소프트웨어 팩스트 링크

### 10.2 견적 결과 페이지 (/quotes/[id])

**3단계 탭 구조**
```
┌──────────────────────────────────────────────────┐
│  🔧 배그 고사양 견적 (배팅 그라운드)  [공유] [저장] │
├─ [💰 최소] [⚖️ 균형] [⚡ 최고] ───────────────────┤
│                                                  │
│ 💰 최소 요구사항 (₩1,450,000)                    │
│  CPU    Intel Core i7-14700K      ₩450,000  ✓  │
│  GPU    NVIDIA RTX 4070 SUPER     ₩600,000  ✓  │
│  MB     ASUS ROG STRIX Z790-F     ₩280,000  ✓  │
│  RAM    Kingston FURY 32GB DDR5   ₩180,000  ⚠️  │
│  SSD    Samsung 990 Pro 1TB       ₩150,000  ✓  │
│  PSU    SEASONIC 850W Gold        ₩180,000  ✓  │
│  Case   NZXT H7 Flow              ₩150,000  ✓  │
│  Cooler be quiet! Dark Rock Pro   ₩150,000  ✓  │
│                                                  │
│  [예상 성능]                                     │
│  • 배그 1080p 고설정: 110fps 예상               │
│  • 총 소비전력: 550W (안전)                      │
│                                                  │
│  [구매 링크]  [부품 교체] [호환성 재검사]        │
│                                                  │
└──────────────────────────────────────────────────┘
```

**호환성 표시**
- ✓ (초록): 완벽 호환
- ⚠️ (노랑): 경고 (성능 다운그레이드 등)
- ✕ (빨강): 비호환 (불가능)
